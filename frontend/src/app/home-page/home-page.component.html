SPRING MARKETPLACE

An online farmers market connecting consumers with locally sourced, fresh produce for a sustainable and vibrant shopping experience.

What is Spring Marketplace?

Spring Marketplace is a Full-Stack Produce Market made using Angular, Java Springboot, and MongoDB as the Database.

This is a 3 person project that aims to mimic modern day online store applications and their features.

Features include Login/Registration w/ hashed passwords, fully customizable profile page, marketplace page, shopping cart, receipt/order viewing, farmers page(s), etc.


Feel free to register and check out the features


How do I use this app?


Using Spring Marketplace as a Buyer

1. Register

2. Start buying on the marketplace

Using Spring Marketplace as a Seller

1. Register

2. Add your info on the profile page

3. List items directly from your profile

Buying Produce

1. Browse whats available on the marketplace

2. Click the item you would like to purchase and add the desired quantity to the cart

3. View your cart by clicking the cart icon

4. Click checkout to purchase, or X to remove specific items from cart
